## Install
Dependencies:
   - Ruby 3.2.2
   - Rails 7.1.3
   - Postgresql

## Run on local
Create database and run migrations (run only once).
```ruby
rails db:create
rails db:migrate
rails db:seed
```

Start local server
```
rails s
```
